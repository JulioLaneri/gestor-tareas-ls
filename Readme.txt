No logré implementar el drag and drop, implementé con botones lo que creo es una liminante al momento de tratar de mover las tareas por las columnas.

Tuve varios incovenientes al tratar de implementar el drag and drop, los cuales no pude solucionar.

Forma de correr:

	*En caso de no tener instalado Node.js descargarlo de https://nodejs.org/es e instalar*

1. Abrir la carpeta del proyecto en Visual Studio Code.
2. Ejecutar el siguiente comando para instalar las dependencias del proyecto: npm install
3. Una vez que la instalación esté completa, ejecutar el siguiente comando para iniciar la aplicación: npm start
